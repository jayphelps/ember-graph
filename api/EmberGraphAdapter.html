EmberGraphAdapter Class

Extends: Adapter
Defined in: src/adapter/ember_graph.js:18

This class acts as a base adapter for synchronous storage forms. Specifically, the LocalStorageAdapter and MemoryAdapter inherit from this class. This class will perform all of the work of updating data and maintaining data integrity, subclasses need only implement the getDatabase and setDatabase methods to create a fully-functioning adapter. This class works with data as a single JSON object that takes the following form:

{
    "records": {
         "type_key": {
             "record_id": {
                 "attr1": "value",
                 "attr2": 5
             }
         }
    },
    "relationships": [{
            "t1": "type_key",
            "i1": "id",
            "n1": "relationship_name",
            "t2": "type_key",
            "i2": "id",
            "n2": "relationship_name",
    }]
}

If you can store the JSON data, then this adapter will ensure complete database integrity, since everything is done is single transactions. You may also override some of the hooks and methods if you wish to customize how the adapter saves or retrieves data.
